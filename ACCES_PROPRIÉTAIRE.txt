    Identifiant  :  simon
    Mot de passe  :  simon

  COMMENT SE CONNECTER (étapes simples) :
  ─────────────────────────────────────
  1. Allez sur https://surf-camp-senegal.vercel.app/admin/
  2. Si votre navigateur demande un accès → entrez simon / simon
     (c'est la protection de démo, pas l'admin)
  3. La page d'admin s'affiche avec 2 champs : identifiant et mot de passe
  4. Entrez simon / simon et cliquez "Se connecter"
  5. Patientez 10 à 20 secondes (connexion GitHub en arrière-plan)
  6. Le tableau de bord CMS s'ouvre → vous pouvez modifier les contenus

  CE QUE VOUS POUVEZ MODIFIER DANS L'ADMIN :
  ───────────────────────────────────────────
  • 🏠 Accueil    : textes, titre SEO, descriptions — en 7 langues
  • 🤙 Surf       : contenu de la page Surfing
  • 🏡 Surf House : contenu de la page hébergement
  • 📅 Réservation: textes de la page booking
  • ❓ FAQ        : questions / réponses
  • 🏝 Île Ngor   : page de présentation de l'île
  • 🗺 Guides     : guides de l'île
  • 📝 Articles   : blog en 7 langues (EN, FR, ES, IT, DE, NL, AR)
                   → créer, modifier, supprimer des articles de blog
  • 👤 Auteurs    : profils des auteurs du blog

  COMMENT SAUVEGARDER LES MODIFICATIONS :
  ────────────────────────────────────────
  → Les modifications sont ENREGISTRÉES AUTOMATIQUEMENT dès que vous tapez.
    Pas besoin d'appuyer sur un bouton Sauvegarder.

  → Pour changer le STATUT de publication d'un article :
    Cliquez sur le bouton "Publié ▼" en haut à droite de l'éditeur
    et sélectionnez l'état souhaité (Brouillon / Publié / En revue).

  → Un badge "✅ Sauvegardé" apparaît brièvement en haut après chaque save.

  ⚠️  IMPORTANT — Comment les changements apparaissent sur le site :
  Les sauvegardes dans l'admin écrivent sur GitHub (dépôt du projet).
  Pour que les modifications soient VISIBLES sur le site live, il faut
  déclencher un rebuild (opération technique effectuée par le développeur).
  → Contactez Simon Azoulay après vos modifications pour déclencher le déploiement.

  DÉCONNEXION :
  → Bouton "Déconnexion" en haut à droite de la barre d'admin noire.

  5. ÉTAPES AVANT LE LANCEMENT PUBLIC
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  □ 1. Configurer le nom de domaine définitif dans Vercel
        (ex : surfcampsenegal.com → pointer les DNS vers Vercel)
  □ 2. Supprimer la variable BASIC_AUTH_PASSWORD dans Vercel
        (retire la protection démo — le site devient public)
  □ 3. Vérifier le formulaire de réservation (email de réception)
  □ 4. Soumettre le sitemap dans Google Search Console
        https://search.google.com/search-console/
        → Ajouter le site → Soumettre https://[domaine]/sitemap.xml
  □ 5. Connecter Google Analytics / Tag Manager si besoin
  □ 6. Ajouter de vraies photos dans les articles de blog
        (remplacer les illustrations AI par des photos réelles du camp)
